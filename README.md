📈 Stock Market Analysis Using MySQL
📌 Project Overview

This project is a complete Stock Market Data Analysis project built using MySQL. The dataset contains historical stock market records along with advanced technical indicators such as RSI, MACD, Moving Averages, Bollinger Bands, Volume Signals, and Trend Indicators.

The main goal of this project is to perform financial analytics, identify market trends, evaluate stock performance, and generate meaningful insights using SQL queries.

This project is ideal for:

1)Beginners learning MySQL through real-world datasets
2)Data Analyst portfolio projects
3)Finance and Stock Market analytics practice
4)SQL interview preparation
5)Technical analysis exploration
